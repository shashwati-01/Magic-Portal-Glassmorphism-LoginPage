# ✨ Magic Portal

# Live Demo
https://shashwati-01.github.io/Magic-Portal-Glassmorphism-LoginPage/

A magical glassmorphism login page featuring animated gradients, glowing effects, and a fairy-inspired responsive interface.

## Features

- Glassmorphism login card
- Animated purple gradient background
- Glowing border effects
- Floating card animation
- Fairy dust effects
- Responsive design
- Email and password input fields
- Ionicons integration

## Technologies Used

- HTML
- CSS
- JavaScript

## How to Run

1. Clone the repository:

```bash
git clone https://github.com/shashwati-01/Magic-Portal-Glassmorphism-Login.git
```

2. Open the project folder.

3. Open `index.html` in a web browser.

## Project Type

Frontend login page design created to explore glassmorphism, CSS animations, gradients, and responsive web design.
